# STEP4 : 로또(수동)

## 기능 요구사항
- [ ] 사용자가 수동으로 추첨 번호를 입력할 수 있도록 해야 한다. / 입력한 금액, 자동 생성 숫자, 수동 생성 번호를 입력하도록 해야 한다.
- 수동으로 로또를 생성
- 수동으로 입력시 사이즈 및 중복번호 확인
- 구매 금액 중 수동 로또 구매를 제외한 자동 로또 생성
